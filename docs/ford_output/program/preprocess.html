Subroutines

typekind_error

Uses

    • utilities_mod

Takes a list of observation type module path names. These modules contain multiple fragments of standard F90 that may be required to implement forward observation operators for DART. The sections are retrieved from the files by this program and inserted into the appropriate blanks in the DEFAULT_obs_def_mod.F90 and DEFAULT_obs_kind_mod.F90 templates. The final obs_def_mod.f90 and obs_kind_mod.f90 that are created contain the default code plus all the code required from the selected observation type modules. Preprocess also inserts the required identifier and string for the corresponding observation kinds (and only those kinds).

Write out the integer declaration lines for kinds
