Parameters (constants):

 maxlevel  - maximum number of depth levels, also maximum
               number of all types of variables
 maxcalc   - maximum number of measured and calculated
               depth dependent variables
 kdim      - number of standard depth levels
 bmiss     - binary missing value marker
 maxtcode  - maximum number of different taxa variable codes
 maxtax    - maximum number of taxa sets
 maxpinf   - maximum number of distinct measured variable
             information codes

Character Arrays:

 cc        - NODC country code
 chars     - WOD character data: 1. originators cruise code,
                                 2. originators station code
 filename  - file name

 nf is the input file indentification number

 Get user input file name from which casts will be
 taken.  Open this file.

SUBROUTINE "WODread": READS IN A SINGLE PROFILE FROM THE ASCII FILE AND STORES THE DATA INTO ARRAYS

Passed Variables:

 nf      - file identification number for input file
 jj      - WOD cast number
 cc      - NODC country code
 icruise - NODC cruise number
 iyear   - year of cast
 month   - month of cast
 iday    - day of cast
 time    - time of cast
 rlat    - latitude of cast
 rlon    - longitude of cast
 levels  - number of depth levels of data
 istdlev - observed (0) or standard (1) levels
 nvar   - number of variables recorded in cast
 ip2(i)  - variable codes of variables in cast
 nsecond - number of secondary header variables
 nbio    - number of biological variables
 isig()  - number of significant figures in (1) latitude, (2) longitude
            and (3) time
 iprec() - precision of (1) latitude, (2) longitude, (3) time
 bmiss   - missing value marker
 ieof    - set to one if end of file has been encountered

Common/Shared Variables and Arrays (see COMMON area of program):

 depth(x)   - depth in meters (x = depth level)
 temp(x,y)  - variable data (x = depth level, y = variable ID = ip2(i))
            ... see also nvar, ip2, istdlev, levels above ...
 sechead(i) - secondary header data (i = secondary header ID = isec(j))
 isec(j)    - secondary header ID (j = #sequence (1st, 2nd, 3rd))
            ... see also nsecond above ...
 bio(i)     - biology header data (i = biol-header ID = ibio(j))
 ibio(j)    - biology header ID (j = #sequence (1st, 2nd, 3rd))
            ... see also nbio above ...
 nbothtot   - number of taxa set / biomass variables
 vtax(i,j)  - taxonomic/biomass array, where j = (1..nbothtot)
               For each entry (j=1..nbothtot), there are vtax(0,j)
               sub-entries.  [Note:  The number of sub-entries is 
               variable for each main entry.]  vtax also holds the
               value of the sub-entries.
 itaxnum(i,j)- taxonomic code or sub-code 
 chars       - WOD character data: 1. originators cruise code,
                                   2. originators station code
 npi        - number of PI codes
 ipi        - Primary Investigator information
              1. primary investigator
              2. variable investigated

 STANDARD LEVELS OR OBSERVED LEVELS
 ----------------------------------

 If this file is on standard levels (istdlev=1), program places the
 standard depths in the depth() array (otherwise, observed depth values 
 were read in and stored above by subroutine WODreadxxxx; where xxxx is
 200X or 1998 depending on the data input format).

 WRITE HEADER INFORMATION TO THE SCREEN
 --------------------------------------

 cc         - country code (a2)
 icruise    - WOD cruise identifier (i8)
 rlat       - latitude (f7.3)
 rlon       - longitude (f8.3)
 iyear      - year (i4)
 month      - month (i2)
 iday       - day (i2)
 time       - time (GMT) (f7.2)
 jj         - WOD cast identifier (i8)
 levels     - number of depth levels measured (i6)

 WRITE CHARACTER DATA TO THE SCREEN
 ----------------------------------

 chars(1)   - Originators cruise identifier
 chars(2)   - Originators station identifier

 WRITE PRIMARY INVESTIGATOR INFORMATION TO THE SCREEN
 ----------------------------------------------------

 npi = number of primary investigator entries
 ipi(1..npi,1)  - PI code
 ipi(1..npi,2)  - variable for which PI was responsible

 WRITE VARIABLE-CODE (column headings) TO THE SCREEN
 ----------------------------------------------------

 nvar          - number of variables (1...nvar)
 ip2(1..nvar)  - variable code for each variable present

 Example:  
   For a cast with just Temperature[1], Oxygen[3], Pressure[25]:

   The variable sequence is ip2(1)=Temperature, ip2(2)=Oxygen, 
      ip2(3)=Pressure

   nvar = 3

   ip2(1) = 1, ip2(2) = 3, ip3(3) = 25


 Note:  If "nvar = 0", biology only cast.

 WRITE DEPTH-DEPENDENT VARIABLE DATA TO THE SCREEN
 --------------------------------------------------

 Print depth (depth(n)), error flags for depth (iderror(n,0)),
 each variable (temp(n,1..nvar)), and error flags for each
 variables (iderror(n,1..nvar))

 PRINT ENTIRE-PROFILE ERROR FLAGS
 ------------------------------------

 WRITE SECONDARY-HEADER INFORMATION TO THE SCREEN
 ---------------------------------------------

 Print the secondary header code (isec(1..nsecond)) and the value
 for that secondary header (sechead(isec(n))).

  WRITE VARIABLE SPECIFIC INFORMATION TO THE SCREEN
  -------------------------------------------------

 WRITE BIOLOGICAL HEADER INFORMATION TO THE SCREEN
 ------------------------------------------

 Print the biology header code (ibio(1..nbio)) and the value
 for that biology header (bio(ibio(n))).

 WRITE TAXA SET/BIOMASS VARIABLE INFORMATION TO THE SCREEN
 ----------------------------------------------------------

 For each set/variable (1..nbothtot), print the set/variable code 
 (ivtax = vtax(1,n)) and each member of that set (1..vtax(0,n)), 
 where the sub-code is itaxnum(n2,n) and the sub-value is vtax(n2,n).
