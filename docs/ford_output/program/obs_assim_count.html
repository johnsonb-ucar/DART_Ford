prints out a quick table of obs types and counts, overall start and stop times, and metadata strings and counts. you can get more info by running the obs_diag program.

right now this program counts up the number of obs for each possible 'DART quality control' value (0-7). (it could also check for a 'posterior ensemble mean' copy and count how many are missing_r8 and how many are not.) it does the former right now.
