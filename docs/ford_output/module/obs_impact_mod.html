This module supports both the obs_impact_tool and reading in the table at runtime for use during the assimilation phase, to alter the impact of observations on the state vector.

these routines build a table(ntypes, nqtys) which can be indexed quickly at runtime. this happens in the assimilation loop so performance matters.


Uses

    • obs_kind_mod
