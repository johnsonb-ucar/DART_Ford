this version of the model_mod is probably more appropriate to start with if you are porting a geophysical model with locations specified in lat/lon and some kind of vertical. it assumes a gridded model. THIS FILE HAS NOT BEEN CONVERTED to read netcdf directly by defining a domain - a list of netcdf variables per input file. THIS WILL BE UPDATED SOON.
